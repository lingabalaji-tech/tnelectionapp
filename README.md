# Tamil Nadu 2026 Voter Facts

Static bilingual voter-information site and dataset for the Tamil Nadu 2026 Assembly election.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Keep the default branch as `main`.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `GitHub Actions`.
5. Push to `main` to trigger the deployment workflow.

The published site is deployed from the `site/` folder through `.github/workflows/pages.yml`.
